# ❤️ CardioSense AI

AI-powered Heart Disease Severity Prediction System built using Machine Learning and Streamlit.

## Features
- Heart Disease Severity Prediction
- XGBoost Machine Learning Model
- Interactive Streamlit Web App
- Dashboard and Visualizations

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit

## Dataset
UCI Heart Disease Dataset
